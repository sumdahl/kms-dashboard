# Plan: Add 500 Internal Server Error Page

## Goal
Create a 500 internal server error HTML page using the existing error.html template, similar to the 404 page.

## Current State
- 404 errors use the error.html template via ErrorTemplate struct.
- 500 errors (panics) currently return JSON via internal_error_response in main.rs.
- Template engine is askama.

## Implementation Steps
1. Modify internal_error_response in src/main.rs to render HTML instead of JSON.
2. Use ErrorTemplate with:
   - code: 500
   - title: "Internal Server Error"
   - message: "Something went wrong on our end. Please try again later."
3. Set Content-Type to "text/html".
4. Render the template to string and include in response body.

## Files to Modify
- src/main.rs: Update internal_error_response function.

## Verification
- Test by triggering a panic or 500 error to ensure HTML page is served.
